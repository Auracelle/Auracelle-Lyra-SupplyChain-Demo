## Overview

Auracelle Lyra is a multi-domain governance assurance simulation platform developed by Auracelle AI Governance Labs. This repository contains a self-contained proprietary demonstration build used for institutional briefings, academic research, and policy engagement.

Lyra evaluates governance assurance capacity across five domains — **AI, Nuclear, Space, Energy, and BioTech** — using a hierarchical scoring architecture. The platform is designed to support:

- Policy document analysis and automated governance scoring
- Cross-domain assurance stress-testing
- Scenario-based governance failure modeling
- Comparative regime analysis

This is a single-file, self-contained HTML simulation. No server, no dependencies, no installation required. Open in any modern browser.

### What this build includes

**Page 1 — E-AGPO · Policy Metrics**
Evaluates governance policy performance using a hierarchical indicator framework. Supports policy document upload for automated scoring (Anthropic API integration) and manual indicator configuration. Includes optimizer and results table.

**Page 2 — E-AGSO · Assurance Stress-Test**
Tests whether governance standards can be verified, audited, and enforced under real-world conditions. Includes:

- Audit Evasion surface modeling
- Assurance NOF configuration
- Continuity of Knowledge (CoK) degradation analysis
- Scenario Library (6 scenarios)

### Scenario Library
